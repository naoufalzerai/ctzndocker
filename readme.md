# ctzndocker

A docker-compose to fire up new [CTZN](https://github.com/pfrazee/ctzn)'s server

## Installation


1. Replace `example.com` with your domain in `entrypoint.sh` and `nginx.conf` and `docker-compose.yml`
2. Replace `example@gmail.com` with your email in `entrypoint.sh`
3. Run

```sh
docker-compose up -d
```
