# ctzndocker

A docker-compose to fire up new [CTZN](https://github.com/pfrazee/ctzn)'s server

## Installation

1. Replace `example.com` with your domain in `entrypoint.sh` and `nginx.conf`
2. Replace `example@gmail.com` with your email in `entrypoint.sh`
3. Run

```sh
git clone https://github.com/naoufalzerai/ctzndocker.git
docker-compose up -d
```
